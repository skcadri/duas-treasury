# A Treasury of Duas
A minimalist web app for reciting daily supplications from the Quran and Sunnah, available at [duas.sohaibq.com](https://duas.sohaibq.com).

## Features
- **Light & Dark Mode**: Handcrafted styling for reading in any environment.
- **Categorized**: Easily switch between All, Quranic, and Hadith duas.
- **Arabic Typography**: Beautiful, traditional fonts for clear recitation.
